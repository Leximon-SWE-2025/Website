Download Release

The releases section of the GitHub repository has precompiled executables of the game for the Windows operating system.

How to Play a Release

  1. Download the desired release as a .zip file.
  2. Unzip the file.
  3. Run the provided .exe file.
  4. Note: All files provided in the release must be kept in the same directory as the executable file.

Build the Game

Required Tools

  • Godot Engine - .NET 4.5.1
  • Godot Engine - .NET 4.5.1 required dependencies

Instructions (Tested on Windows 11)

  1. Download and install the correct version of Godot Engine.
    • If required, download and install any other required software as listed on the download page for Godot Engine.
  2. Clone the repository from the desired commit.
  3. Run the Godot executable.
  4. Import the project in the Godot editor by selecting Import from the project list and using the folder for the repository.
  5. Open the game.
  6. Go to Run the Project in the Godot Editor or Export the project as desired (instructions below).

Run the Project in the Godot Editor

  1. Build the project by clicking the Build Project button (top right of the UI).
  2. Wait for the build to complete.
  3. Run the project by clicking the Run Project button (located directly right of the Build Project button).

Export the Project

  1. Ensure Export templates are installed:
    • Select: Editor -> Manage Export Templates.
    • Ensure best available mirror is selected.
    • Press Download and Install.
    • Wait for download to complete.
    • Close Export Template Manager.
  2. Select Project -> Export.
  3. Add desired export template(s).
  4. Click Export All.
  5. Click Release.
  6. The exported game will be in the folder chosen when setting up the export template(s).

Note: all files produced by export must be kept in the same directory as the executable file.

Play The Game - Controls

Basic Controls

  • Use w, a, s, d or the arrow keys to move.
  • Use f to open the inventory to view player info and word info.
  • Use e when close to an enemy to engage in a battle.
  • Use escape to close the inventory, leave a battle, or open the exit menu.

Inventory and Battle UI

  • In the inventory: Any word can be clicked on with the mouse to view its types as well as its definitions.
  • In the battle menu: A word can be clicked to open a menu to choose how to use it.
    • The options are to attack, defend, or cancel selection.
  • The help text can be clicked on to view the word's definition(s).

Battle Progression Notes

  • When either combatant runs out of health, the battle is over.
  • Every 3 battles won, the Player's type is randomized.
